# 30 Days of Agentic APIs

30 days, 30 demos, one AI developer platform per day. I want to understand the
agent infrastructure landscape by actually building on all of it — so I am.

Each day: a working demo, a short writeup of what I learned and where the DX
hurt, and a Twitter thread. Follow along at [@georgepickett](https://x.com/georgepickett).

realtime, observability and evals, deployment and runtime infra.
